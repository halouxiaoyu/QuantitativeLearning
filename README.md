# AI量化交易学习平台

一个专为初学者设计的AI量化交易学习平台，采用分层架构设计，支持完整的量化交易流程：从数据获取到模型训练，从回测分析到未来预测。

## 🚀 项目特色

- **分层架构设计**：清晰的数据层、特征层、模型层、预测层分离
- **完整交易流程**：数据获取 → 特征工程 → 模型训练 → 回测分析 → 历史验证 → 未来预测
- **Web可视化界面**：直观的操作界面，实时进度显示
- **单股票分析**：支持输入单个股票代码进行深度分析
- **机器学习集成**：集成随机森林、XGBoost等高性能算法
- **技术指标丰富**：包含移动平均、RSI、MACD等常用技术指标
- **特征优化**：智能特征选择，18个核心特征，减少噪音提高性能
- **标签优化**：使用0.1%涨跌幅阈值，预测第二天涨跌，实现完美数据平衡
- **实用预测**：第二天预测支持每日交易决策，提高资金利用效率

## 🏗️ 系统架构

```
AI量化交易学习平台
├── 数据层 (Data Layer)
│   ├── 数据获取与清洗 (DataManager)
│   └── 本地数据存储
├── 特征层 (Feature Layer)
│   ├── 技术指标构建 (FeatureEngineer) - 18个核心特征
│   ├── 特征选择优化 (FeatureSelector)
│   └── 特征数据管理
├── 模型层 (Model Layer)
│   ├── 模型训练与保存 (ModelTrainer) - 随机森林/XGBoost
│   └── 交叉验证与评估
├── 预测层 (Prediction Layer)
│   ├── 历史预测验证 (HistoricalPredictor)
│   └── 未来预测 (FuturePredictor)
├── 回测层 (Backtest Layer)
│   ├── 策略回测 (BacktestAnalyzer)
│   └── 性能指标分析
└── Web界面层
    ├── Flask后端API
    └── 响应式前端界面
```

## 📋 功能模块

### 1. 数据获取与清洗
- 支持多数据源：akshare、baostock
- 自动数据清洗和标准化
- 本地数据存储管理
- 数据状态监控
- **新增**：单股票数据下载和验证

### 2. 特征工程 (已优化)
- **核心特征**：18个最重要的技术指标
- 技术指标构建：
  - 移动平均线 (MA5, MA10, MA20)
  - 价格比率 (MA5_ratio, MA10_ratio, MA20_ratio)
  - 指数移动平均线 (EMA12, EMA26)
  - MACD指标 (MACD_dif, MACD_dea)
  - 成交量指标 (Volume_MA5, Volume_ratio)
  - 布林带 (BB_upper, BB_lower)
  - 价格波动 (High_low_ratio, Volatility_10)
  - 趋势指标 (Trend_5)
- **特征优化**：18个核心特征，减少噪音，提高性能
- 特征数据持久化
- 特征质量检查

### 3. 特征选择与优化 (新增)
- **智能特征选择**：基于特征重要性的自动选择
- **特征数量优化**：支持8、10、12、15、18个特征的选择（推荐18个）
- **性能对比**：不同特征数量的性能评估
- **最佳实践**：推荐18个特征的最佳配置

### 4. 模型训练与保存 (已优化)
- **算法选择**：随机森林、XGBoost
- **标签优化**：0.1%涨跌幅阈值，预测第二天涨跌，实现完美数据平衡
- 时间序列数据分割
- 交叉验证评估
- 模型性能指标
- 模型持久化存储

### 5. 历史预测验证
- 使用未参与训练的数据验证模型
- 预测准确率评估
- 模型泛化能力测试
- 验证结果分析

### 6. 未来预测
- 基于最新特征预测第二天涨跌
- 生成交易信号 (BUY/SELL/HOLD)
- 置信度评估
- 预测结果保存
- **实用优势**：第二天预测支持每日交易决策，提高资金周转效率

### 7. 回测分析
- 历史策略回测
- 性能指标计算：
  - 总收益率
  - 年化收益率
  - 最大回撤
  - 夏普比率
  - 胜率统计
- 交易记录分析

## 📖 使用指南

### 快速开始

1. **数据获取**
   - 输入单个股票代码（如：002639）
   - 设置时间范围（建议2020-2025）
   - 点击"开始下载"

2. **特征构建**
   - 系统自动构建18个核心特征
   - 包含技术指标和标签生成
   - 点击"开始构建特征"

3. **模型训练**
   - 选择算法：随机森林（推荐）或XGBoost
   - 设置交叉验证折数（推荐5折）
   - 点击"开始训练模型"

4. **回测分析**
   - 设置回测参数
   - 点击"开始回测"

5. **历史验证**
   - 设置验证时间范围
   - 点击"开始历史验证"

6. **未来预测**
   - 输入股票代码
   - 设置预测天数（1-5天）
   - 点击"开始未来预测"

### 工作流程

```
数据获取 → 特征工程(18个特征) → 模型训练(随机森林/XGBoost) → 回测分析 → 历史验证 → 未来预测
    ↓              ↓                    ↓              ↓         ↓         ↓
  原始数据 → 核心技术指标+标签 → 高性能模型 → 策略回测 → 模型验证 → 交易信号
```

## 🚨 注意事项

1. **数据质量**：确保数据源稳定，数据质量良好
2. **模型风险**：机器学习模型存在过拟合风险，建议定期重新训练
3. **回测偏差**：历史回测结果不代表未来表现
4. **资金管理**：实际交易请做好风险控制和资金管理
5. **特征选择**：18个特征已经过优化，不建议随意增减
6. **标签阈值**：0.1%阈值已经过测试，可根据市场情况微调
7. **预测周期**：第二天预测已经过验证，支持每日交易决策
